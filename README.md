# 小鹤双拼练习

小鹤双拼（双拼）在线练习工具。

## 功能

- 短句输入练习，错误处原地重试
- 随机诗词接口出题，按整首诗逐句练习
- 实时统计正确率、速度、等级
- 虚拟键盘实时提示按键
- 深色/浅色主题切换
- 练习结束后查看详细错字记录

## 本地运行

需要 Node.js 20.17 或更高版本。

```bash
npm install
npm run dev
```

## 测试

```bash
npm test
```

## 构建部署

```bash
npm run build
# dist/ 目录部署到静态托管即可
```

## 技术栈

Vite + React
